# Actividad Momento 2

Este proyecto corresponde a la **presentación de la actividad Momento 2**.  
Incluye archivos en **HTML, CSS y JavaScript** utilizados para mostrar la información y cumplir con los requisitos de la actividad.

---

## 🎯 Objetivo de la actividad
Presentar el contenido solicitado para el Momento 2 de manera clara, organizada e interactiva utilizando una página web sencilla.

---

## 📂 Contenido del proyecto
- `index.html` → Página principal de la actividad  
- `styles.css` → Estilos del diseño  
- `script.js` → Lógica y comportamiento en JavaScript  
- `.gitignore` → Archivos/carpetas que no se suben al repositorio  
- `README.md` → Descripción general del proyecto

---

## ▶️ Cómo visualizar la presentación
1. Descarga o clona este repositorio.  
2. Abre el archivo `index.html` en cualquier navegador (Chrome, Edge, Firefox).  

No requiere instalaciones adicionales.
